.. _installation:

Installing
======================

Для установки последней версии из репозитория PyPI

.. code-block:: console

    pip install sgen

Installing with git
-------------------

The project is hosted at https://github.com/Apels1nA/sgen and can be installed
using git:

.. code-block:: console

    git clone git@github.com:Apels1nA/sgen.git
    cd sgen
    python setup.py install
